# Demographic Data Analyzer

A Python data analysis project built using Pandas as part of the FreeCodeCamp Data Analysis with Python certification.

## Features

* Calculates the percentage of people with a Bachelor's degree.
* Compares the percentage of people with advanced education (Bachelor's, Master's, Doctorate) earning more than 50K versus those without advanced education.
* Determines the minimum number of work hours per week.
* Calculates the percentage of people working the minimum number of hours who earn more than 50K.
* Identifies the country with the highest percentage of people earning more than 50K.
* Computes the percentage of high-income earners for each country.
* Finds the most popular occupation among high-income earners in India.
* Performs demographic and income analysis using real census data.
* Utilizes Pandas operations such as filtering, indexing, aggregation, and value counting.

## Technologies Used

* Python
* Pandas

## Project Structure

* `demographic_data_analyzer.py` – Main analysis logic
* `main.py` – Runs and displays results
* `adult.data.csv` – Census dataset

## Learning Outcomes

* Data filtering with Boolean indexing
* Working with Series and DataFrames
* Aggregation and statistical analysis
* Using `value_counts()`, `mean()`,  and indexing operations
* Real-world data analysis with Pandas
